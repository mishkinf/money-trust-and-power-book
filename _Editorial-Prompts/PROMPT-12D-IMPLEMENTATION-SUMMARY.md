## Key Principles Across All Prompts

### 1. No New Research Required (Option A)
All prompts offer "Option A" strategies that work with existing material:
- Develop content already present in chapters
- Add detail to stories/examples already mentioned
- Expand on themes already introduced
- No new factual claims requiring sources

### 2. Research-Required Alternatives (Option B)
Each expansion prompt also offers "Option B" for those willing to invest research time:
- Additional historical examples
- More biographical detail
- Deeper institutional analysis
- **Only pursue if you want to invest 6-12 hours in research**

### 3. Voice and Tone Consistency
All prompts include detailed guidance on maintaining:
- Sentence rhythm patterns (short/medium/long)
- Narrative style (dramatic but precise)
- Explanatory approach (clear, concrete, analogies)
- Human story integration (visceral, specific)
- Contemporary connections (natural, not forced)

### 4. Quality Checks
Each prompt includes verification checklists:
- Word count targets
- Tone consistency
- Argument integrity
- Flow and transitions
- No orphaned references
