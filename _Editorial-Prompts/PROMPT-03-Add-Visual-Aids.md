**Purpose:** Show the cascade of deflationary crises 1925-1936 as countries stayed on/returned to gold standard, making the pattern of failure visually obvious.

## Technical Specifications

### Format Convention
All visual aids use this format in manuscript:

```markdown
### [VISUAL-AID-X.X]

**[Title of Visual]**

[Description of what visual should show, or in some cases, actual table content if simple]

**PLACEHOLDER:** [Detailed specifications for designer]

---
```

Some existing visual aids include actual tables in markdown (e.g., VISUAL-AID-13.2); others provide specifications for professional design (e.g., VISUAL-AID-11.1 Triffin arithmetic diagram).

### Placement Guidelines

**Place visual aids:**
- AFTER concept is introduced in text
- BEFORE reader needs to understand it to follow subsequent argument
- Where attention might flag (dense passages benefit from visual break)

**Example from Chapter 13:**
- Productive capacity principle explained in text (pages 15-21)
- [VISUAL-AID-13.1] appears immediately after (page 21)
- Subsequent analysis of 2008 vs. 2021 assumes reader has grasped diagram

### Citation Requirements

Each new visual aid MUST:
1. Draw data only from existing chapter content OR flagged research files
2. Have all factual claims cited in chapter's sources.md file
3. If data comes from Research folder, include citation to original source (not just "Research/filename.md")

**Example citation chain:**
- Visual aid shows "Venezuela oil production fell from 3.2M barrels/day (2013) to 700K (2020)"
- Chapter 13 sources.md must cite: "BP Statistical Review of World Energy 2021" or equivalent
- Research file `Hyperinflation - Real collapse precedes money printing.md` may be intermediate source but ultimate citation must be primary data

## Quality Standards

### 1. Accessibility
- No unexplained jargon in visual aid titles or labels
- Legends/keys explain all symbols
- Color choices work in grayscale (for print editions)

### 2. Accuracy
- Every number must have source
- Dates verified against chapter content
- Causal arrows or connections reflect text argument, not speculation

### 3. Clarity
- Visual should be comprehensible in 30 seconds
- One clear takeaway (e.g., "Triffin dilemma was mathematically inevitable")
- Not cluttered with excessive data

### 4. Integration
- Text references visual aid explicitly ("Visual Aid 10.1 shows the cascade...")
- Visual aid placement interrupts text at logical point, not mid-paragraph

## Research Gap Flagging Protocol

If creating any visual aid reveals that existing sources are insufficient:

**DO NOT:**
- Invent plausible-sounding data
- Cite "common knowledge" without source
- Simplify to point of inaccuracy

**INSTEAD:**
- Note specific gap: "RESEARCH NEEDED: Annual US gold reserves 1950-1971 (currently have only 1950, 1960, 1971 snapshots)"
- Assess whether gap is critical: Can visual work with less granular data? Alternative format?
- Propose either (a) simplified version using available data or (b) defer visual aid until research completed

**Example:**

Two options: (1) Create table with these five data points (adequate for showing trend), or (2) Research question: Can we obtain annual data from IMF International Financial Statistics or Federal Reserve archives?"
